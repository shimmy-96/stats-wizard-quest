# 🧪 Stats Wizard Quest

מדריך תרגול אינטראקטיבי ל**מבוא לסטטיסטיקה ב׳** — לומדים למבחן כמו משחק.

אפליקציית דף-אחד (single-file HTML), עובדת לגמרי אופליין, בעברית (RTL). אין שרת, אין ספריות חיצוניות, אין מעקב — ההתקדמות נשמרת מקומית בדפדפן שלך בלבד.

## מה יש בפנים
- **9 נושאים**: ריענון סטטיסטיקה א׳, מבחן Z, עוצמה וטעויות, רווח סמך, מבחני t, מתאם ורגרסיה, ANOVA חד/דו-כיווני, חי-בריבוע.
- לכל נושא: תקציר, תיאוריה, נוסחאות, הדגמה אינטראקטיבית, דוגמה מודרכת, ובוחן.
- **דף נוסחאות** חי, **לוחות סטטיסטיים מלאים** (Z · t · F · χ²) עם בוחן קריאה.
- **מאמן פלטי Excel**, עץ החלטה "איזה מבחן לבחור", מילון מושגים, כרטיסיות שינון, מבחנים מלאים, ואבחון טעויות אישי.

## שימוש
פותחים את `index.html` בכל דפדפן — זהו. עובד גם בלי אינטרנט, גם בטלפון.

## סטטוס
כלי תרגול **לא-רשמי** ללימוד עצמי. הוכן ע״י שימי (ניהול טכנולוגיה, בר-אילן), מבוסס על חומרי הקורס של ד״ר גלית כהן-בוטבול. השאלות והדוגמאות מקוריות (נתונים גנריים).
